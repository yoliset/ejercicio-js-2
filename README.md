# Ejercicio2

Inicio : Promedio de 3 Numeros

## Ingreso datos
- Numero1
- Numero2
- Numero3
- Suma
- Promedio

## Proceso

- Pedir Numero1
- Pedir Numero2
- Pedir Numero3
- Sumar Numero1 y Numero2 y Numero3 y almacenar en Suma
- Dividir Suma entre 3 y almacenar en Promedio
- Mostrar Promedio
